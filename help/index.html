Электронная имитация параметров КПД

Программа позволяет имитировать параметры КПД и отображает отвертную реакцию блока.

Начало работы

Подключите БУ к ИПК-3 (используйте кабель с автоматическим управлением питания БУ), включите питание блока Перейдите в режим поездки нажатием кнопки «ДАТА/П» на БУ. Запустите программу. После загрузки программы выберите нужный блок из выпадающего списка в верхней части окна. После запуска программы с БУ считываются условно-постоянные признаки, их значения используются для имитации параметров. Если программа не смогла их получить (по CAN), будет выведена ошибка, а признаки будут считаны из внутреннего файла конфигурации (значения которые были записаны на этом типу БУ при последней проверке).
Для управления питанием КПД и сигналом TURT используйте чекбокс с соответствующим названием. Для БУ-4 вместо TURT отображается чекбокс «Режим обслуживания» и выполняет ввод пин-кода (вход в режим обслуживания для изменения настроек БУ-4), выход из режима - перезагрузка блока.
В нижей части окна программы есть статусная строка, на которой будут отображаться статус выполнения имитации, ошибки или подсказки.

Установка условно-постоянных признаков (УПП)

Нажмие кнопку «УПП». В открывшемся окне будет выведен список УПП соответствующий БУ, выбранному ранее из выпадающего списка. Значения УПП считываются с блока при загрузке окна и доступны к редактированию.
Значения УПП сохраненные во внутреннем файле конфигурации (значения которые были записаны на этот тип БУ последний раз) можно получить нажав кнопку «Сохранённые УПП» в нижей области окна.
Текущие значение УПП с блока можно получить нажав кнопку «УПП БУ».
Скорректируйте значения УПП, проверьте, что значения находятся в допустимых пределах. Установите чекбокс «Управлять питание» (если блок заранее небыл переведен в режим обслуживания), нажмите кнопку «Записать».

Данные CAN

Программа получает данные по CAN. Основные параметры для выбранного типа БУ сформированы в список в правой части окна. Если на данный момент программа не получает данные о параметре в его строке будет стоять «-». Если на данный момен в блоке установлна ошибка, её код будет отображаться в этом списке. Кликните по строчке с кодом ошибки для того, чтобы в статусной строке отобразилась ее расшифровка.

Имитация движения

Для установки скорости и ускорения введите параметр в соответствующее окно и нажмите «Enter». Если чекбокс «Раздельное управление» установлен, скорость и ускорение можно установить разную для канала 1 и 2. Если чекбокс не установлен, то значение вводимое в окне «Канал 1», дублируется в окне «Канал 2» и наоборот и устанавливается сразу на оба канала. Скорость, установленную на блоке в данный момент можно увидить в поле «Данные CAN» для кабины 1 и кабины 2. Если программа не получает значение в данный момент, в строке отображается «-».
Для установки напраления движения используйте переключатель с пометками «Вперёд» и «Назад». При смене направления в поле «Данные CAN» в строках «Движение вперёд» и «Движение назад» меняется значение.
Параметры имитации движения (число зубьев ДУП и диаметры бандажей колёс) используются те, что установлены на БУ в текущий момент. Если их нужно изменить, установите УПП БУ.

Имитация пути

Введите дистанцию, нажмите кнопку «Пуск», установите скорость, текущая позиция (пройденное расстояние) будет выведено в поле с пометкой «Текущая». Для остановки имитации введите 0 в поле дистанции, нажмите «Пуск». Общая пройденная дистанция с момента включения блока отображается в поле «Данные CAN».

Имитация давления

Установите давление в диапазоне допустимом для данного канала, нажмите «Enter» для установки. Установленное значение отображается в поле «Данные CAN».

Двоичные сигналы

Для установки двоичного выхода БУ установите соответствующий чекбокс в поле «Вых. БУ». Некоторые установленные на БУ сигналы (тяга, АЛС, ЭПК) можно контролировать в поле «Данные CAN».
Для установки кода рельсовой цепи установите соответствующую кнопку переключатель в поле «Коды РЦ». Установленный сигнал можно контролировать в поле «Данные CAN» в строке «Сигнал ИФ». В поле «Реле превышения уставок» отображаются входные сигналы БУ. Их установка ни к чему не приведёт.
